# 14-amaliy mashg'ulot - Dasturlash

## Loyihaning maqsadi
Bu amaliy mashg‘ulotda Python’da bir nechta funksiyalar yaratildi va ular bitta asosiy faylda chaqirildi. Maqsad — funksiyalarni modullashtirish, kodni tartibli va qayta foydalanish mumkin bo‘lishini ta’minlash.

## Bajarilgan ishlar
- **juft_toq.py** – sonning juft yoki toqligini aniqlovchi funksiya.
- **matn_uzunlik.py** – matn uzunligini hisoblovchi funksiya.
- **teskari_matn.py** – matnni teskari chiqaruvchi funksiya.
- **musbat_sonlar.py** – musbat sonlarni ajratib beruvchi funksiya.
- **orta_qiymat.py** – o‘rtacha qiymatni hisoblovchi funksiya.
- **parol_tekshir.py** – parolni tekshiruvchi funksiya.
- **son_tekshir.py** – sonni tekshiruvchi funksiya.
- **main.py** – yuqoridagi barcha funksiyalarni chaqiruvchi asosiy fayl.

## Tushunchalar
- Funksiyalarni alohida fayllarda yozish va ularni chaqirish orqali kodni modullashtirishni o‘rgandim.
- Python’da `import` orqali boshqa fayldagi funksiyalarni ishlatishni mustahkamlab oldim.
- Git buyruqlari yordamida kodni GitHub’ga joylashni amalda sinab ko‘rdim.
